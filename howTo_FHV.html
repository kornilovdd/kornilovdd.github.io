Как пользоватся конфигуратором FHV7H

Для использования конфигуратора вам потребуется сделующая информация по задаче:

1. Размеры области (ROI - region of interest) и высоту упаковки в слое (для послойной агрегации кодов).
2. Информация о размере кода - линейный размер в мм и размер в ячейках.
3. Количество слоев (для послойной агрегации кодов).

Порядок заполнения формы:

1. Введите параметры "Длина", "Ширина" и "Высота" слоя (параметр "Высота" используется для послойной агрегации).
2. Выберите объектив из выпадающего списка. Обратите вниманеие, что при выборе C-mount объективов параметр "Количество слоев" будет заблокирован.

Возможность автоматической фокусировки есть только на объективах FHV-LEM.

3. Выберите тип кода в выпадающем списке - QR или DataMatrix.
4. Выберите размерность кода в ячейках и линейный размер кода в выпадающих списках.
5. Выберите количество слоев продукта (доступно только при выборе объективов FHV-LEM).
6. Нажмите кнопку "Подобрать".

Вывод данных

После проведения рассчетов отобразится таблица с названием модели камеры,
значений Field Of View по оси X и оси Y. Обратите внимание, что значение параметра "Ширина"
может отличаться от заданного. В случае, если рассчетное значение меньше заданного, поле "Ширина" в форме ввода будет выделено красным.
Далее выводится параметр "Разрешение" - количество пикселей матрицы камеры на один элемент кода, измеряется в "ppe" - pixels per element. Если это значение ниже 3, то процент считывания существенно снижается. В этом случае результата рассчетов представлено не будет. Пересмотрите размер области по оси Y в меньшую сторону. Справа выводится параметр "WD" - Working Distance. Значение рассчитывается с учетом наличия нескольких слоев, если был сделан такой выбор и обозначает расстояние установки именно до нижнего слоя продукта.
